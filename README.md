# Competitor Low-Star Review Audit

A public validation page for a small research service:

> Read public competitor complaints before choosing a SaaS, App, plugin, or automation wedge.

This repository hosts a free sample audit for Shopify returns and exchange apps. The sample reviews public app pages, public low-star reviews, public pricing, Shopify community discussions, and Shopify help documentation.

## Pages

- `index.html` - public audit landing page
- `sample-report.html` - full sample report
- `index.zh-CN.md` - Simplified Chinese companion copy for the landing page
- `sample-report.zh-CN.md` - Simplified Chinese companion copy for the sample report

## What The Sample Covers

The sample looks at:

- AfterShip Returns
- Loop Returns
- Return Prime
- ReturnGO
- Shopify native returns and exchanges

The core verdict:

> The Shopify returns category is not closed, but I would not start by building another full returns platform.

More practical wedges include pre-launch app-selection checks, low-volume merchant workflows, Markets / multi-currency compatibility reviews, and narrow low-star review audits for App categories.

## Boundary

This audit only uses public information.

It does not use customer logins, private dashboards, order data, payment data, cookies, production tokens, refunds, disputes, KYC, or legal commitments.

## Validation Goal

This is a 7-day validation asset. The goal is to test whether indie builders, SaaS founders, Shopify App developers, Chrome extension builders, or automation service providers would ask for a similar audit before building or repositioning a product.

Success signals:

- 1 paid order
- 3 clear inquiries
- 5 target users asking for their own category or competitors
